** 時不変性
- 定義
  #+begin_quote
  入力の時刻が \(s\) ずれた場合，出力も \(s\) だけずれる性質
  #+end_quote
  - 時間が経過してもフィルタの性質は変わらない

** 線形時不変フィルタの数学的表現
- フィルタの積分表現
  #+begin_quote
  #+begin_src latex
    \begin{equation}
      g(t)=\int_{-\infty}^{\infty} f(s)h(t{-}s)ds
    \end{equation}
  #+end_src
  #+end_quote
- インパルス応答
  #+begin_quote
  #+begin_src latex
    \begin{equation}
      h(t)
      =\int_{-\infty}^{\infty}\delta(s)h(t-s)ds
      =\int_{-\infty}^{\infty}h(s)\delta(t-s)ds
    \end{equation}
  #+end_src
  #+end_quote
  - \(h(t)\) はフィルタに \(\delta(t)\) を入力した時の出力でもある

** Fourier変換による表現
- 時間領域では畳み込み積分
  #+begin_quote
  #+begin_src latex
    \begin{equation}
      g(t)
      =\int_{-\infty}^{\infty}h(t-s)f(s)ds
      = h{*}f(t)
    \end{equation}
  #+end_src
  #+end_quote
  - 周波数領域では関数の積
    #+begin_quote
    #+begin_src latex
      \begin{equation}
	\hat{g}(\omega)
	=
	\sqrt{2\pi}\cdot\hat{h}(\omega)\cdot\hat{f}(\omega)
      \end{equation}
    #+end_src
    #+end_quote
  - フィルタの機能は周波数毎の振幅と位相の変換

** 因果的フィルタ
- 定義
  #+begin_quote
  #+begin_src latex
    \begin{equation}
      h(t)=0\;(t<0)
    \end{equation}
  #+end_src
  #+end_quote
  - 時刻0にインパルスが入力される前には何も出力がされない
- 因果的フィルタの畳み込み
  #+begin_quote
  時刻 \(t\) での出力は
  時刻 \(t\) 以前での入力のみにより定まる
  #+begin_src latex
    \begin{equation}
      g(t)=\int_{-\infty}^{t}f(s)h(t-s)ds
    \end{equation}
  #+end_src
  #+end_quote

- 関数 \(t e^{-\frac{t^2}{2}}\) の Fourier 変換
  #+begin_quote
  #+begin_src latex
    \begin{equation}
      \mathcal{F}[f'](\omega)
      =i\omega\hat{f}(\omega),\quad
      \mathcal{F}^{-1}[\hat{f}'](t)
      =-itf(t)
    \end{equation}
  #+end_src
  を利用すればよい
  #+begin_src latex
    \begin{equation}
      t e^{-\frac{t^2}{2}}
      =
      (-e^{-\frac{t^2}{2}})'
    \end{equation}
  #+end_src
  であるから
  #+begin_src latex
    \begin{equation}
      \mathcal{F}[t e^{-\frac{t^2}{2}}](\omega)
      =-i\omega \mathcal{F}[e^{-\frac{t^2}{2}}](\omega)
      =-i\omega e^{-\frac{\omega^{2}}{2}}
    \end{equation}
  #+end_src
  #+end_quote

#+reveal: split
- 関数 \(\Xi_{(-1,1)}(t)\) の Fourier 変換
  #+begin_quote
  #+begin_src latex
    \begin{equation}
      \mathcal{F}[\Xi_{(-1,1)}](\omega)
      =
      \sqrt{\frac{2}{\pi}}\frac{\sin(\omega)}{\omega}
    \end{equation}
  #+end_src
  #+end_quote

#+reveal: split
- 関数 \((\sin(\omega)/\omega)^{2}\) の 逆 Fourier 変換
  #+begin_quote
  畳み込みを利用
  #+begin_src latex
    \begin{equation}
      \mathcal{F}[h{*}f](\omega)
      =\sqrt{2\pi}\cdot\hat{h}(\omega)\cdot\hat{f}(\omega)
    \end{equation}
  #+end_src
  #+begin_src latex
    \begin{align}
      \mathcal{F}^{-1}[\hat{f}^{2}](t)
      &=\frac{1}{\sqrt{2\pi}}f{*}f(t)\\
      &=\frac{1}{\sqrt{2\pi}}
	\sqrt{\frac{\pi}{2}}^{2}\Xi_{(-1,1)}{*}\Xi_{(-1,1)}(t)\\
      &=\sqrt{\frac{\pi}{8}}\Xi_{(-1,1)}{*}\Xi_{(-1,1)}(t)
    \end{align}
  #+end_src
  #+end_quote

** COMMENT 解答例
:PROPERTIES:
:reveal_background: #fef4f4
:END:
- 時間領域
  #+begin_quote
  以下の微分方程式が成り立つ
  #+begin_src latex
    \begin{align}
      V_{in}(t)
      &=RI(t)+V_{out}(t)\\
      I(t)
      &=C\frac{d}{dt}V_{out}(t)
    \end{align}
  #+end_src
  整理して
  #+begin_src latex
    \begin{equation}
      V_{in}(t)
      =CR\frac{d}{dt}V_{out}(t)
      +V_{out}(t)
    \end{equation}
  #+end_src
  #+end_quote

#+reveal: split
- 周波数領域
  #+begin_quote
  時間領域の結果を Fourier 変換して
  #+begin_src latex
    \begin{equation}
      \hat{V}_{in}(\omega)
      =i\omega CR\hat{V}_{out}(\omega)
      +\hat{V}_{out}(\omega)
    \end{equation}
  #+end_src
  入出力の関係を見直して
  #+begin_src latex
    \begin{equation}
      \hat{V}_{out}(\omega)
      =
      \frac{1}{i\omega CR+1}
      \hat{V}_{in}(\omega)
    \end{equation}
  #+end_src
  #+end_quote

#+reveal: split
- フィルタとしての関係
  #+begin_quote
  #+begin_src latex
    \begin{equation}
      \hat{g}(\omega)
      =
      \sqrt{2\pi}\cdot\hat{h}(\omega)\cdot\hat{f}(\omega)
    \end{equation}
  #+end_src
  と比較して
  #+begin_src latex
    \begin{align}
      \hat{f}(\omega)
      &=\hat{V}_{in}(\omega)\\
      \hat{g}(\omega)
      &=\hat{V}_{out}(\omega)\\
      \sqrt{2\pi}\cdot\hat{h}(\omega)
      &=\frac{1}{i\omega CR+1}
	=\frac{1}{iCR}\frac{1}{\omega -i/CR}
    \end{align}
  #+end_src
  #+end_quote
  - 周波数依存性について考察せよ


* 逆 Fourier 変換による解析
** インパルス応答
- 逆 Fourier 変換
  #+begin_quote
  式を見易くするため \(a=1/CR\) とおく
  #+begin_src latex
    \begin{align}
      h(t)
      &=\mathcal{F}^{-1}[\hat{h}](t)\\
      &=\frac{1}{\sqrt{2\pi}}\int_{-\infty}^{\infty} \hat{h}(\omega)e^{i\omega t}d\omega\\
      &=\frac{a}{2\pi i}\int_{-\infty}^{\infty} \frac{e^{i\omega t}}{\omega-ia}d\omega
    \end{align}
  #+end_src
  #+end_quote

#+reveal: split
- 複素積分の積分路
  - \(t>0\) のとき， \(ia\) を囲む上半平面
  - \(t<0\) のとき， 下半平面

#+reveal: split
- 留数定理
  #+begin_quote
  積分路が孤立特異点 \(c\) を含むとき以下が成り立つ
  #+begin_src latex
    \begin{equation}
      \frac{1}{2\pi i}\oint f(z)dz
      =\mathrm{Res}_{z=c}f(z)
      =\lim_{z\to c}(z-c)f(z)
    \end{equation}
  #+end_src
  #+end_quote

  #+begin_src latex

      h(t)
      =

	a e^{-at}=\frac{1}{CR}e^{-\frac{t}{CR}},&t>0\\
	0,&t<0
      \end{cases}
    \end{equation}
  #+end_src

** COMMENT 解答例
:PROPERTIES:
:reveal_background: #fef4f4
:END:
- \(0 < t \le \Delta\)
  #+begin_quote
  #+begin_src latex
    \begin{equation}
      \frac{1}{\Delta}
      =CR\frac{d}{dt}V_{out}(t)
      +V_{out}(t)
    \end{equation}
  #+end_src
  \(U(t)=V_{out}(t)-1/\Delta\) と置くと
  #+begin_src latex
    \begin{equation}
      \frac{d}{dt}U(t)=-\frac{1}{CR}U(t)
    \end{equation}
  #+end_src
  よって
  #+begin_src latex
    \begin{equation}
      V_{out}(t)=a e^{-\frac{t}{CR}}+\frac{1}{\Delta}         
    \end{equation}
  #+end_src
  #+end_quote

#+reveal: split
- 初期条件 \(V_{out}(0)=0\)
  #+begin_quote
  #+begin_src latex
    \begin{equation}
      0=a+\frac{1}{\Delta}
    \end{equation}
  #+end_src
  よって
  #+begin_src latex
    \begin{equation}
      V_{out}(t)=\frac{1}{\Delta}(1-e^{-\frac{t}{CR}})
    \end{equation}
  #+end_src
  #+end_quote

#+reveal: split
- \(\Delta < t\)
  #+begin_quote
  #+begin_src latex
    \begin{equation}
      0
      =CR\frac{d}{dt}V_{out}(t)
      +V_{out}(t)
    \end{equation}
  #+end_src
  よって
  #+begin_src latex
    \begin{equation}
      V_{out}(t)=a e^{-\frac{1}{CR}}
    \end{equation}
  #+end_src
  #+end_quote

#+reveal: split
- 初期条件 \(V_{out}(\Delta)=1/\Delta\cdot(1-e^{-\frac{\Delta}{CR}})\)
  #+begin_quote
  #+begin_src latex
    \begin{equation}
      \frac{1}{\Delta}(1-e^{-\frac{\Delta}{CR}})
      =a e^{-\frac{\Delta}{CR}}
    \end{equation}
  #+end_src
  よって
  #+begin_src latex
    \begin{equation}
      V_{out}(t)=\frac{1}{\Delta}(e^{\frac{\Delta}{CR}}-1)e^{-\frac{t}{CR}}
    \end{equation}
  #+end_src
  #+end_quote

#+reveal: split
- \(\Delta\to 0\)
  #+begin_quote
  #+begin_src latex
    \begin{equation}
      \lim_{\Delta\to0}
      \frac{1}{\Delta}(e^{\frac{\Delta}{CR}}-1)
      =\frac{1}{CR}
    \end{equation}
  #+end_src
  よって
  #+begin_src latex
    \begin{equation}
      V_{out}(t)=\frac{1}{CR}e^{-\frac{t}{CR}}\quad(t>0)
    \end{equation}
  #+end_src
  #+end_quote
